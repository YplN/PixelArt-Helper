Comment m'utiliser?

  • Rentrez les coordonnées de départ de votre région (coin en haut à gauche) et celles de fin (coin en bas à droite).
  • Si vous le souhaitez, changez la taille de l'affichage des pixels.
  • Choisir le fichier correspondant au dessin voulu. Note : si l'image est entièrement blanche, changez une valeur quelconque dans les coordoonées, cela devrait résoudre le problème (vous pouvez ensuite revenir à la valeur initiale).
  • Passer la souris sur un pixel pour faire apparaître les coordonnées de celui-ci et son code hexadécimal correspondant.
  • Si vous cliquez sur un pixel, le code hexadécimal sera copié dans votre presse-papier.
  • (BETA) En cliquant sur Comparer, cela affiche les différences de pixels entre l'image et le drapeau réel (merci à CoDaTi#3827 pour son API ! ❤ https://codati.ovh/doc/)
